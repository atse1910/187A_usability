01. Visibility



“The system should always keep users informed about what is going on, through appropriate feedback within reasonable time..” --Nielsen
#1. It provides title on each page. It is useful to help people know what page they are looking at.
#2. This website provides navigation bar at the top of each page.
       The top navigation helps to move to other directory easily.
#3. This menu seems not neccesary.

Which means
This website provides title of each directory. It is very helpful to indicate to the user what page they are on. In addition it preserves top navigation bar at the top of each page. It is perfectly helpful with looking for other resources. However, the menu #3. seems unneccesary. Because, the information could be found the first page of website.
